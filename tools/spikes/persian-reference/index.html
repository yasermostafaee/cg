5. ZWNJ — می‌خواهم vs میخواهم

With ZWNJ (correct):
می‌خواهم بروم خانه

می ends in detached form; خواهم starts in initial form.

Without ZWNJ (wrong):
میخواهم بروم خانه

می joins to خواهم with medial form — incorrect for this verb.

Pair test:
نمی‌توانم   vs   نمیتوانم
Common compounds:
می‌گویند، می‌دانم، می‌بینیم

PASS = the with-ZWNJ versions match standard Persian typography references. ZWNJ is U+200C; visible only as a zero-width separator that breaks the joining behavior.

6. ZWJ (rare, but specified)

می‌رود vs می‍رود

ZWJ (U+200D) forces joining where it wouldn't normally apply. Rarely used; included for completeness.
